Change Log

  • Fixed an issue where special characters were encoded in the email subject
  • Fixed an issue that caused an error message to be shown when run with jQuery 1.4 (the version used in DNN 5.3+)
  • Fixed an issue where QueryString parameters were not being included in the link sent in the email
  • Added a token reference definition list to the module settings page
  • Added fields to the module settings page that allow the customization of the email subject and body
  • Added fields to the module settings page that allow for setting the "from" email address
  • Added fields to the module settings page that allow for including CC and BCC recipients in the email

Known Issues

  • ActiveModules' Active Forums module cannot post topics when Tell-a-Friend is on the page
