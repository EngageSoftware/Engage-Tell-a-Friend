Features

  • Allows site visitors to send an email to the current page or a specific URL
  • Allows for customization of all text through Localization
  • Provides a responsive and intuitive end-user interface
  • Uses a standards-based XHTML/CSS approach
  • Provides performance and usability improvements by using an AJAX approach

Known Issues

  • Close button tooltip text on modal popup is not localizable
  • After the first time that the modal popup is displayed, validation errors are not displayed
